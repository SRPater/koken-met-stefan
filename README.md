# Koken Met Stefan

A personal repository for aggregating and organizing recipes from various sources into a single, unified interface.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Routing:** React Router
- **State Management:** Context API
- **Formatting:** Prettier

## Setup
1. `npm install`
2. `npm run dev`
3. `npm run format`
